Inset

The inset value for the border property is used to create an "inset" or pressed effect on the border of an element. It gives the appearance of the border being pushed into the element.

Solid

This style creates a solid, continuous line for the border. It's the most common border style used.

Dashed

This style creates a series of short dashes for the border. Provides a dashed appearance to the border.

Outset

This style creates a 3D outset border effect. Gives the appearance of popping out from the page.

Groove

This style creates a 3D grooved border effect. Gives the appearance of a carved groove.

Ridge

This style creates a 3D ridged border effect. Gives the appearance of a raised ridge.

Double-line

This style creates two parallel lines for the border. Creates a bold, double-line effect for the border.

Dotted

This style creates a series of dots for the border. Gives a dotted appearance to the border.
